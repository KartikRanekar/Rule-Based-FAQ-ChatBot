# Institute FAQ Chatbot

A simple rule-based chatbot built with Streamlit to answer common institute-related questions.

## Features
- Answers 15+ common questions (Fees, Timings, Contact, etc.)
- Simple keyword-based matching.
- User-friendly chat interface.

## Setup Instructions

1.  **Install Python**: Ensure you have Python installed on your system.
2.  **Install Dependencies**:
    Open a terminal in this folder and run:
    ```bash
    pip install -r requirements.txt
    ```

## How to Run

Run the application using the following command:
```bash
streamlit run app.py
```

## Supported Topics
You can ask questions related to:
- Timings, Holidays
- Fees, Admissions
- Courses, Results, Exams
- Contact, Address
- Library, Hostel, Transport
- Placements, Sports, Events
